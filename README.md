# To-Do List App

## Description
The **To-Do List App** is a simple, web-based task management application built using **HTML, CSS, and JavaScript**.  
It allows users to **add, edit, mark as completed, and delete tasks**, with **persistent storage** using the browser's local storage.  

This app is fully **responsive** and works on desktops, tablets, and mobile devices.

---

## Features
- Add new tasks
- Edit existing tasks
- Mark tasks as completed
- Delete tasks
- Data persistence with **local storage**
- Responsive design

---

## Technologies Used
- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**

---

## Installation / How to Run
1. **Download or clone** the repository:
    ```bash
    git clone https://github.com/amartgit/Todo_List.git
    ```
2. Navigate to the project folder:
    ```bash
    cd Todo_List
    ```
3. Open the `index.html` file in any modern web browser.
4. Start adding tasks and managing your to-do list.

## Usage
1. **Add Task:** Type a task in the input field and press `Enter` or click `Add`.  
2. **Edit Task:** Click the edit icon next to a task to update it.  
3. **Complete Task:** Click the checkbox to mark a task as done.  
4. **Delete Task:** Click the trash icon to remove a task permanently.  
5. **Persistent Storage:** Tasks remain even after refreshing the page, thanks to **local storage**.
